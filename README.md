# REST Countries API Project

## Overview
This project uses the REST Countries API to display information about countries around the world. Users can search for countries and filter them by region.

## Features
- View all countries from the API
- Search for countries by name
- Filter countries by region
- Combined search and region filtering
- Dynamic rendering of country data

## Technologies Used
- HTML
- CSS
- JavaScript
- REST Countries API

## How It Works
The application fetches country data from the REST Countries API and stores it in a variable. A function is used to display the countries on the page. Search and region filters update the displayed countries dynamically without reloading the page.

## Deployment
The project is deployed using GitHub Pages.

## What I Learned
I learned how to work with APIs, handle asynchronous JavaScript using fetch, and manipulate the DOM to dynamically display data. I also learned how to structure my code using functions and improve it by refactoring.
