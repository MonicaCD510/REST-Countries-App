Reflection

For this project, I built a REST Countries application using HTML, CSS, and JavaScript. The app fetches data from an API and displays country information, with search and filtering features to make the data interactive.

I ran into many challenges every step of the way. I spent a great deal of Stime debugging and testing different approaches, which helped me better understand how JavaScript interacts with the DOM and how to troubleshoot issues more effectively.

I was able to complete the core functionality, including fetching data, rendering it dynamically, and allowing users to search and filter results. This project helped strengthen my understanding of working with APIs and structuring JavaScript code in a more organized way.

If I were to continue improving this project, I would focus on enhancing the user experience and refining some of the interactive features. Overall, this project helped me build confidence in applying JavaScript to create dynamic, data-driven applications.
